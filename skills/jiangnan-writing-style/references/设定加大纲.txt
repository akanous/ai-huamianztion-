一、角色设定
主角：浅羽司
| 项目 | 内容 |
|------|------|
| 年龄 | 16岁，高中一年级 |
| 身份 | 穿越者（前世：实验室猝死的大学生，父母双亡） |
| 灵力 | 异常庞大，穿越后无法控制；高中二次觉醒后眼睛变为蓝色带波纹，平时戴深棕色美瞳掩饰 |
| 能力 | 指尖释放极细青蓝色灵光束；本能看穿怪物术式核心；夜视能力+灵力感知 |
| 术式进化 | Ch1 失控→Ch2 结界收缩不到2米→Ch3 半径半米（被动）→Ch4 分流（结界+攻击同步） |
| 性格 | 情感迟钝（不是冷漠，是真的不知道某些场合该说什么）；生活自律（姐姐教的），唯一短板是做饭难吃；学业天才（年级前十） |
| 过去 | 6-7岁灵力觉醒伴随高烧，被周围人排斥；父母死于大灾变 |
| 生活 | 与姐姐分居两地，姐姐负担全部费用；戴黑色露指手套作为「控制」的仪式 |
| 核心动力 | 保护姐姐（减少她的负担）；保护真唯（从饭团到战斗） |
| 关键台词 | 「不是缺不缺的问题。是她不需要每一分都给我。」 |
女主角A（正宫）：早川真唯
| 项目 | 内容 |
|------|------|
| 年龄 | 16岁，高中一年级，与司同校不同班 |
| 身份 | 偏远小神社（早川神社）的巫女 |
| 灵力 | 极弱——幼年被姐姐理央逐步吞噬，几近于无 |
| 术式 | 基础结界术（封锁结界）、净化术 |
| 性格 | 三层结构：表面元气冒失、成绩稀烂；中层极度努力（无天赋靠时间堆）；底层对姐姐感情复杂 |
| 过去 | 原姓中野，父母死于大灾变后改随母姓「早川」以表示独立；十一岁独自离开家 |
| 核心台词 | 「三年」——别人三分钟学会的东西她花了三年，但她做到了 |
| 感情线 | 细水长流，和司互帮互助，终成正果 |
| 角色弧线 | Ch1 独自战斗→Ch2 接纳司→Ch3 组队（助手）→Ch4 封锁结界透支→Ch5 面对姐姐电话 |
| 特色道具 | 破旧笔记本（灵力笔记）、弹珠汽水、便当盒 |
女主角B（神子）：中野理央
| 项目 | 内容 |
|------|------|
| 年龄 | 17岁，高中二年级，比司和真唯大一级 |
| 身份 | 大神社（中野神社）继承人，被称为「神子」 |
| 灵力 | 极强——吞噬了妹妹（真唯）的灵力 + 自身天赋 |
| 性格 | 三层结构：表冷淡完美、遥不可及；中感情白痴、行动力极强、敢做敢想；底怀着对妹妹的愧疚与无法言说的歉意 |
| 过去 | 保留了父姓「中野」——象征继承、责任、无法放下的过去；父母让理央吞噬真唯灵力 |
| 感情线 | 中期发力（Vol 7 表白），行动派，感情笨拙但一往无前，比妹妹更快一步 |
| 登场时机 | Vol 2 正式登场；Vol 1 Ch9 仅走廊擦肩而过 |
姐姐（锚）：浅羽美月
| 项目 | 内容 |
|------|------|
| 年龄 | 23岁，普通上班族（静冈文员），高中毕业 |
| 性格 | 温柔、天然呆、在弟弟面前装可靠其实自己也在硬撑 |
| 定位 | 表世界的「锚」——司人性保持的锚点和生活支柱 |
| 对司的期望 | 上大学、谈恋爱、享受青春 |
| 保护规则 | 她的安全是全作底线——受伤必须是全作最重大转折点 |
| Ch5关键 | 通过电话察觉司和真唯的关系——签名改为「放心了」 |
| 隐藏属性 | 文员——每天从措辞模糊的邮件中找真相，能一眼看穿「有效就可以」式隐瞒 |
关键配角：佐伯健一郎
| 项目 | 内容 |
|------|------|
| 年龄 | 约30岁出头 |
| 身份 | 内阁府特务，权限高于一般灵能者协会 |
| 外貌 | 深灰色风衣，银框眼镜，站姿放松 |
| 动机 | 寻找六年前在旧铁道灵力泄漏事故中失踪的妹妹（时年十一岁） |
| 与主线关系 | 监控鹰栖诚也六年；提供河床主脉情报；要求司和真唯制造鹰栖的失控以突破其记忆壁 |
| 隐藏设定 | 夜祓系统内权限高到可覆盖系统推送——可能与匿名私信者为同一人 |
| 对真唯 | 点破她封锁结界上限——「崩塌和突破的边界是一样的」 |
关键反派（第一卷）：鹰栖诚也
| 项目 | 内容 |
|------|------|
| 年龄 | 约30岁，前内阁府研究机构成员 |
| 身份 | 原灵力残留处理方向研究员，六年前旧铁道事故后主动失联 |
| 能力 | 灵力扫描术式、碎片的制造与拼接（用错误方法试图中和河床主脉） |
| 外貌 | 白大褂、白手套、无线接收器、头发整齐 |
| 状态 | 正在失控——记忆被自己用术式封锁，实验周期被司/真唯打乱 |
| 标签 | 写「鹰」字贴在便利贴和器材上——提醒自己姓鹰栖的备忘录 |
| 核心矛盾 | 用错误的方程式解一道对的题：想抑制河床，方法导致次生怪物爆发 |
二、反派组织
高天原
| 层级 | 成员 | 说明 |
|------|------|------|
| 表向名称 | 神代文化财团 | 表面做民俗文化保护和神社修缮，手续齐全，「干净到像是被人整理过」 |
| 顶层 | 三柱神 | 天之御中主神、高御产巢日神、神产巢日神——日本造化三神 |

| 登场节奏 | Vol 2中后期 | 首次有名字的登场；Vol 1 仅出现身份不明的白衣研究员（无Logo、无组织名） |
三、世界观核心设定
| 项目 | 内容 |
|------|------|
| 世界结构 | 表世界（普通人日常）+ 里世界（灵能者/巫女/怪物） |
| 灵力体系 | 先天灵力 + 后天术式；眼睛是灵力表现窗口 |
| 灵压规则 | 对所有人产生压迫感，普通人本能恐惧，里世界人因习惯而能承受 |
| 怪物来源 | 日本都市传说与妖怪为蓝本；Vol 1核心=人造碎片怪物 |
| 夜祓App | 里世界危机响应平台：任务派遣、灵力地图、预测算法、报酬结算 |
| 报酬体系 | 灰级1-2万日元/黄级6-10万日元/橙级16-30万日元 |
| 大灾变 | 多年前反派释放大规模怪物，司和两女主角的父母均死于该事件 |
| 旧铁道河床 | 大灾变级灵力泄漏的残余——六十年沉积物=整条旧铁道所有怪物的源头 |

第一卷主线：司从灵力失控→控制半米→分流双线→组队战斗；揭露旧铁道=鹰栖的实验场；建立「天才×凡人努力」的搭档关系
第二卷：学校日常 + 第二个怪物 + 理央登场
- 理央正式登场，校园内与真唯形成「遥不可及的神子 vs 不起眼的小巫女」对照
- 第二个怪物类型：与旧铁道不同的都市传说系
- 神代文化财团名字首次在对话中被提及
- 司和真唯的关系在校园日常中进一步深化
- 佐伯继续提供情报支持
第三卷：高天原名字首次出现 + 第一个干部
- 高天原组织名称正式被提及——可能通过佐伯或某个事件
- 星辰众第一个干部登场（定位：中Boss，卷末战）
- 旧铁道河床危机进一步发酵——鹰栖失控？
- 真唯灵力极限继续被挑战
第四卷：暑假合宿篇
- 经典轻小说合宿：可能神社集训或灵能者训练营
- 理央和真唯第一次较长时间同框（仍不知道彼此关系）
- 司的控制力再上一个台阶
- 感情线升温——日常中的细水长流
第五卷：校园祭篇
- 校园祭=表里世界交汇的舞台
- 高天原可能在校园祭中安排某种行动
- 真唯/理央的关系线索埋更多伏笔
- 姐姐可能来东京参加校园祭——碰撞
第六卷：黑暗加深 + 真唯受伤
- 高天原行动升级，直接威胁到主角群
- 真唯受伤——第一卷之后最大危机
- 司的愤怒第一次突破情感迟钝的界限
- 理央在真唯受伤后的反应——读者开始怀疑两人关系
第七卷：中期转折 + 理央表白
- 理央比真唯更快一步表白——「明明是我先来的」名场面蓄势
- 司面对表白——情感迟钝者的第一次被动觉醒
- 高天原的某个大计划开始露出全貌
- 星辰众更多干部登场
第八卷：神子线升温 + 姐姐危机预警
- 理央的感情线深化——行动派不断推进
- 姐姐遇到某种与里世界相关的危机——为第九卷的重大转折预警
- 司必须在「保护姐姐」和「里世界责任」之间做选择
- 真唯对理央的敌意从「看不惯」升级为「某种模糊的不安」
第九卷：姐妹真相
- 真唯和理央的姐妹关系完全揭露
- 灵力吞噬的真相——父母的选择，理央的愧疚，真唯的选择
- 「明明是我先来的」——名场面正式引爆
- 姐妹关系决裂或重组的第一步
第十卷：姐妹关系重组
- 真唯和理央从对立走向某种新的关系
- 司在中间的立场——不是选择谁，是帮两人找到彼此
- 高天原趁乱发动攻击
- 姐姐的危机正式爆发（保护规则触发——全作最重大转折点）
第十一卷：反击
- 主角群从被动转为主动
- 针对高天原的反击计划
- 星辰众干部逐个击破
- 司的能力达到新高度
第十二卷：牺牲
- 重大牺牲——可能是配角或某个干部洗白后的牺牲
- 高天原的三柱神开始亲自下场
- 世界观的终极真相逐渐揭开
第十三卷：觉醒
- 司的灵力完全觉醒——前世记忆和灵力本源
- 真唯的灵力极限突破（佐伯预言「崩塌和突破的边界是一样的」）
- 理央面对自己的罪与赎
第十四卷：最终决战前夜
- 备战阶段——集结所有势力
- 大灾变级别的危机重演威胁
- 感情线收束
第十五卷：终局
- 最终决战
- 高天原覆灭 / 三柱神结局
- 三个「十一岁」的收束：真唯、理央、佐伯的妹妹
- 表世界与里世界的新平衡
- 司和真唯的结局
- 最后一幕——呼应第一卷第一页
六、关键数字与意象
| 元素 | 含义 |
|------|------|
| 三年 | 真唯学会别人三分钟学会的事——凡人努力的核心象征 |
| 十一岁 | 真唯离家、佐伯妹妹失踪、理央吞噬灵力——三个女人的命运交汇点 |
| 半径半米 | 司的控制力原点——从保护自己到保护两个人的距离 |
| 饭团 | 司表达关心的方式——不需要语言的情感载体 |
| 弹珠汽水 | 真唯的能量来源——每次透支后被司托着罐底 |
| 旧铁道 | 过去的创伤——大灾变的伤痕，鹰栖的执念，司的战场 |
| 美瞳 | 司在两个世界之间切换的仪式——戴上是普通人，摘下是灵能者 |
| 蓝色眼睛 | 灵力表现——从失控的暴风雨到受控的银色慢涌 |
