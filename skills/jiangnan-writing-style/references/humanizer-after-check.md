# 交付后检查：AI痕迹扫描速查表

完成一章后，按 jiangnan-writing-style 的 Step 4 做一次 AI 痕迹检视。

## 中文写作中常见 AI 痕迹

| 模式 | 表现形式 | 修改方向 |
|------|----------|----------|
| 破折号过度 | 一章出现 8+ 次「——」 | 换成句号断句或自然停顿，保留 2-3 处 |
| 比喻堆叠 | 同一段用 2-3 个「像……一样」 | 砍到 1 个，保留最有力的 |
| 「不是……是……」对称 | 反复用「不是X，是Y」句型 | 只留一处做节奏变化，其余改直接陈述 |
| 句式节奏工整 | 每段 2-4 行，速度均匀 | 插短句（1-2 字成段），让呼吸有变化 |
| 对话标签过多 | 每个对话都带「他说」「她说」 | 去掉一半，上下文能看出谁在说 |
| 虚词填充 | 「大约」「像是」「仿佛」反复出现 | 删掉大部分，减少精确填充感 |
| 分段过于均匀 | 所有段落长度相近 | 穿插极短段和长段 |
| 每段带铺垫 | 先写环境再进核心 | 砍掉第一句铺垫，直接进画面 |
| 结尾升华 | 用总结性句子收尾 | 换成具体画面或动作 |

## 流程

写作（jiangnan-writing-style）→ docx 输出 → AI痕迹扫描 → 修改 → 最终交付
